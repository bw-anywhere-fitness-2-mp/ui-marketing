ANYWHERE FITNESS

How It Works

Instructors


Anywhere Fitness allows instructors to work with a diverse client base and set up 1 on 1
meetings or classes from a variety of locations.


Instructors can:


  • Create a class for Clients to join
  • Schedule 1 on 1 meetings with a Client
  • Build up good ratings to increase client Base
  • Request and Process Payments
  • Create virtual "Punch Passes" for each type of class offered

Clients


Anywhere fitness allows you to create a flexible work out schedule, tenured to your needs. It is designed so that clients can work-out on demand, or "on-location", whether that be a virtual workout with your instructor, at a park, or at a local workout facility.


Clients can easily find information on classes -


  • Location
  • Class size
  • Start time and Duration
  • Reschedule or Cancel an appointment or reservation
  • Choose an Instructor based on rating and personal preference
